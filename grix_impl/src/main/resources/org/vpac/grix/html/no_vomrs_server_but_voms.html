This VO only provides a VOMS server...

...not a VOMRS server on top of it.

This means that Grix could not help you request membership for this VO. Also, Grix can't show you detailed information about your membership once you are a member.

But, since you see this page, Grix detected that you are already a member of this VO and therefor you can create voms proxies for this VO.
