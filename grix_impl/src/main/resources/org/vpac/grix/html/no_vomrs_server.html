Can not connect to this VOMRS server.

This can have several reasons:

  • This machine either does not have a network connection.

  • This machine has got a network connection but the neccessary port to connect to the VOMRS server is blocked by a firewall.

  • Most likely though, this error only means that the VO you are trying to contact does not provide a VOMRS server but chances are that it provides a VOMS server.

    Because of this, Grix can't help you requesting membership (and manage this membership once you are a member) for this VO. You can do it via the VO's webinterface, though. Click the URL on the bottom and subscribe for the VO. If you can't connect to the site you probably did not load your certificate into your browser.
